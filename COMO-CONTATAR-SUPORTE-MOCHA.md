# Como Contatar o Suporte da Mocha

## Opções para Obter Ajuda

### 1. Interface de Chat do Mocha
- Procure por um ícone de chat ou mensagem na interface do Mocha
- Geralmente localizado no canto inferior direito da tela
- Permite conversar diretamente com a equipe de suporte

### 2. Menu de Configurações
- Clique no menu de configurações (ícone de engrenagem ⚙️)
- Procure por opções como:
  - "Suporte"
  - "Ajuda"
  - "Contato"
  - "Help & Support"

### 3. Menu do Usuário
- Clique no seu avatar/foto no canto superior direito
- Procure por opções de suporte ou contato

### 4. Email Direto
Se não encontrar as opções acima, tente:
- support@mocha.app
- hello@mocha.app
- help@mocha.app

### 5. Documentação
- Visite: https://mocha.app/docs
- Ou procure por "Documentation" no menu principal

## O Que Solicitar ao Suporte

Quando entrar em contato, seja específico:

```
Assunto: Solicitação de Exportação Completa do Projeto

Olá,

Preciso exportar todo o código do meu projeto VetSalesPro 
(https://vetsalespro.mocha.app) para entregar ao cliente.

Poderiam me ajudar a:
1. Fazer download de todos os arquivos do projeto em um arquivo ZIP
2. Exportar o schema completo do banco de dados
3. Obter as configurações de ambiente e secrets (sem os valores sensíveis)

Meu projeto ID: [Procure nas configurações do projeto]

Obrigado!
```

## Informações Úteis para o Suporte

Tenha em mãos:
- URL do projeto: https://vetsalespro.mocha.app
- Nome do projeto: VetSalesPro
- Quantidade de arquivos: ~55 arquivos de código
- Tecnologias: React, TypeScript, Hono, Cloudflare Workers, D1

## Alternativa: Verificar Integrações

1. Vá em Configurações do Projeto
2. Procure por "Integrations" ou "Integrações"
3. Verifique se há opção de conectar com GitHub
4. Se houver, conecte seu repositório GitHub
5. O código será sincronizado automaticamente

## Dica Importante

Se você é desenvolvedor e vai entregar o projeto ao cliente:
- Solicite acesso de administrador ao projeto
- Peça para transferir ownership para uma conta do cliente
- Ou solicite exportação completa em formato ZIP

## Precisa de Mais Ajuda?

Se nenhuma dessas opções funcionar:
- Verifique se há um botão "?" na interface
- Procure por "Feedback" ou "Report an Issue"
- Entre em contato através das redes sociais do Mocha
